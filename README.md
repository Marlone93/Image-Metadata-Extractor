Setup:
1. Install Python
2. Install pillow & exifread via. pip install pillow exifread
3. Copy the Repo and put it into a Folder.
4. Run the .py
5. Enter the Path. Example: C:\Users\Username\Test.jpg

Full Supported:
jpg/jpeg
tiff/tif

Half Supported:
png
bmp
gif
ico
webp
